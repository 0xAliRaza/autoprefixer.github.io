Автопрефиксер онлайн (autoprefixer css online)

Добавляем нужные вендорные префиксы и удаляем ненужные в вашем CSS

/* Тут будет ваш CSS только с нужными префиксами */

Префиксы проставляются в соответствии с последними данными о поддержке CSS свойств в браузерах на основе сайта caniuse. Вы можете выбрать под какой браузер вам нужны префиксы. Внизу левой колонки есть фильтр, с его синтаксисом можно ознакомиться у меня на сайте ymatuhin.ru.
